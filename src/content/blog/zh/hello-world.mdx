# Hello, world

做了好几年 Chrome 扩展，写了不少散落在各处的笔记。这个周末终于抽出时间，把属于自己的互联网门面搭起来：**hi.kyriewen.cn**。

## 这里有什么

- **产品矩阵** —— 我在持续迭代的所有 Chrome 扩展和 Web 应用。
- **博客** —— 技术笔记、独立开发者心得，中英双语。
- **Now 页** —— 我**现在**在关注什么，不是去年在做什么。
- **数据看板** —— Build in Public：GitHub Star、Chrome 用户、订阅数，公开透明。
- **留言墙** —— GitHub 登录即可留言，打个招呼吧。

## 为什么还要做个人站

> _互联网上唯一真正属于你的那一块地，就是你自己的域名。_

微博会删，即刻会改版，公众号会限流，Twitter 一夜改叫 X。但一个部署在我自己域名下的 Next.js 站，是我自己的。

## 接下来

继续写，继续发布。数据看板会做得足够诚实 —— 包括那些长得慢的数字。
